>>## 如何将bbs.sql导入MySQL数据库：
>>1，创建数据库bbs：create database bbs;
>>
>>2，进入bbs数据库：use bbs;
>>
>>3，将bbs.sql导入bbs数据库：source 你的路径\bbs.sql;
>>### 技术栈
>>**jsp** **servlet** **javabean** **mysql** **html** **css** **javascript** **jQuery**
